# Tasks
## Task #1

Cтраница, содержащая функцию, реализованную на JavaScript со следующим функционалом. 
Функция принимает массив строк в качестве аргумента, выделяет на странице прямоугольные области одинакового размера (размер областей одинаков и задается как константа) и размещает элементы массива в этих областях. 
Т.е. 0-й элемент массива в первой области, 1-й во второй и т.д.
